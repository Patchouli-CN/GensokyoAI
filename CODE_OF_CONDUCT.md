# Code of Conduct

GensokyoAI 希望参与者在 Issue、Pull Request、讨论和文档中保持尊重、友好和建设性。

## 我们提倡

- 尊重不同经验和背景的参与者。
- 聚焦技术问题、事实和改进方案。
- 接受并给予建设性反馈。
- 在发现问题时帮助复现、定位和修复。

## 我们不接受

- 人身攻击、骚扰、威胁、辱骂或歧视性言论。
- 发布他人隐私信息、凭证、联系方式或未公开资料。
- 恶意刷屏、引战或破坏性行为。
- 提交恶意代码、后门或试图绕过项目安全边界的内容。

## 处理方式

维护者可根据情况提醒、警告、隐藏、关闭或删除不当内容，并在必要时限制参与。

## 报告方式

如需报告问题，可通过项目 Issue 或维护者公开渠道联系。请尽量附上链接、截图或日志。
