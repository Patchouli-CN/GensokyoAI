# Developer Certificate of Origin

本项目采用简洁版 Developer Certificate of Origin（DCO）来确认每位贡献者有权提交其贡献。

向 GensokyoAI 提交代码、文档、测试、配置、角色文件、示例或其他内容，即表示你确认：

1. 该贡献由你创作，或你有权按照本项目许可证提交。
2. 该贡献不侵犯第三方版权、商标、专利、商业秘密或其他权利。
3. 如果包含第三方内容，你已确认其许可与本项目兼容。
4. 你同意该贡献可按照 Apache License 2.0 被使用、修改、复制、合并、发布和分发。

## Signed-off-by

推荐在提交时加上：

```bash
git commit -s -m "描述你的变更"
```

这将生成 `Signed-off-by:` 声明，用于标记你确认上述内容。
